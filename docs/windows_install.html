Instalace aplikace Nástroje OP JAK (Windows)

Uživateli doručujeme dva soubory: tento HTML návod a install_windows.ps1. Skript po spuštění stáhne hotový obsah z větve windows-install, zkontroluje potřebné programy a nainstaluje aplikaci do zvolené složky včetně zástupce na ploše. Uživatel tedy nepotřebuje žádné znalosti GitHubu ani příkazů Git.

Tip: Celý proces běží lokálně na vašem počítači. Žádná data se neodesílají mimo zařízení a aplikace vyžaduje MS Excel pro správné zpracování šablon.

1. Co obdržíte

  • Instalační skript install_windows.ps1
  • Tento návod s popisem kroků

2. Předpoklady (instaluje uživatel ručně)

Nainstalujte následující software (práva administrátora jsou doporučená):

  • Python 3.11 až 3.13 (64-bit) – python.org/downloads (při instalaci zaškrtněte Add Python to PATH)
  • Node.js 18 LTS – nodejs.org (nainstaluje i npm)
  • Git for Windows – git-scm.com
  • Microsoft Office s Excelem – 2019+ nebo Microsoft 365, povolte makra pro xlwings
  • Microsoft Visual C++ Redistributable 2015-2022 – download (nutné pro některé Node moduly)
  • PowerShell 5.1+ / Windows 10+ – součást systému, doporučené spouštět skript v režimu „Spustit jako správce“

3. Příprava složky a spuštění skriptu

  1. Rozbalte doručený balíček do složky, kde chcete mít instalační skripty (např. C:\OPJAK\installer).
  2. Otevřete PowerShell jako správce (Start → napište „PowerShell“ → pravým tlačítkem „Spustit jako správce“).
  3. Přejděte do složky se skriptem, např.: 
    cd C:\OPJAK\installer
  4. Pokud spouštíte skripty poprvé, povolte je: 
    Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
  5. Spusťte instalaci (výchozí cesta C:\OPJAK): 
    .\install_windows.ps1
    nebo zadejte vlastní umístění: 
    .\install_windows.ps1 -InstallRoot "D:\Aplikace\OPJAK"

4. Co skript provede

Skript provede:

  1. Kontrolu dostupnosti Pythonu, pipu, Node.js, npm a Gitu.
  2. Zobrazení seznamu chybějících komponent (bez pokračování instalace).
  3. Stažení/aktualizaci větve windows-install do vybrané instalační složky.
  4. Vytvoření a aktivaci virtuálního prostředí (venv), instalaci Python balíčků z requirements-windows.txt.
  5. Instalaci Node modulů v režimu npm install --production.
  6. Vytvoření konfigurace (soubor .env, výstupní složky).
  7. Kontrolní test (python test_students_16plus.py).
  8. Vytvoření zástupce Nástroje OP JAK na ploše (odkaz na start-app.bat).

5. První spuštění

  • Dvojklik na zástupce nebo běh start-app.bat.
  • Backend (Python) se spustí na pozadí automaticky.
  • V prvním spuštění potvrďte oprávnění firewallu, pokud bude vyžádáno.

6. Aktualizace aplikace

Po instalaci najdete ve složce aplikace adresář scripts.

  1. Otevřete PowerShell a přejděte do složky s aplikací (např. C:\OPJAK\electron_app).
  2. Spusťte: 
    .\scripts\update_windows.ps1
    (volitelně s -InstallRoot, pokud instalace není v C:\OPJAK).
  3. Skript stáhne aktuální verzi z větve windows-install, obnoví Python/Node balíčky a spustí kontrolní test.

7. Odinstalace

  • Ukončete aplikaci.
  • Smažte složku instalace a zástupce.
  • Případně odstraňte virtuální prostředí venv.
Potíže? Otevřete složku logs a přiložte soubory backend.log a electron.log kolegovi, který poskytuje podporu. Do zprávy napište, ve kterém kroku se instalace zastavila.
