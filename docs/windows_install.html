📦 Instalace aplikace Nástroje OP JAK (Windows)

Jednoduchý návod pro instalaci aplikace na Windows PC. Celý proces trvá cca 10-15 minut a vše běží lokálně na vašem počítači.

✅ Bezpečnost: Aplikace běží kompletně offline na vašem počítači. Žádná data se neodesílají na internet.

1️⃣ Co budete potřebovat

Před instalací si připravte tyto programy:

Program Verze Odkaz ke stažení Poznámka
Python 3.11 – 3.13 (64-bit) python.org/downloads ⚠️ Zaškrtněte "Add Python to PATH"
Node.js 18 LTS nebo novější nodejs.org Nainstaluje i npm
Git for Windows Nejnovější git-scm.com Pro stažení a aktualizace
Microsoft Office 2019+ nebo Microsoft 365 Váš systém Nutný Excel pro zpracování
VC++ Redistributable 2015-2022 (x64) Stáhnout Závislost pro Node moduly
⚠️ Důležité: Při instalaci Pythonu musíte zaškrtnout volbu "Add Python to PATH", jinak instalace nebude fungovat!

2️⃣ Instalace – 3 jednoduché kroky

1Stáhněte instalátor

Stáhněte si soubor install.bat z GitHubu:

⬇️ Stáhnout install.bat

Nebo navštivte: github.com/maxparez/electron_app/releases/latest

2Spusťte instalaci

Dvojklikem spusťte stažený soubor install.bat

  • Skript zkontroluje, zda máte nainstalované všechny potřebné programy
  • Pokud něco chybí, zobrazí se seznam s odkazy ke stažení
  • Po dokončení kontrol spustí automatickou instalaci

3Hotovo!

Po dokončení instalace najdete na ploše zástupce "Nástroje OP JAK"

  • Dvojklik spustí aplikaci
  • Při prvním spuštění potvrďte oprávnění firewallu (pokud se zobrazí)
  • Aplikace je připravena k použití

3️⃣ Co instalační skript udělá

Instalátor automaticky provede tyto kroky:

  1. Kontrola závislostí – ověří Python, Node.js, npm, Git
  2. Stažení aplikace – naklonuje z GitHubu do C:\OPJAK\electron_app
  3. Python prostředí – vytvoří virtuální prostředí a nainstaluje knihovny
  4. Node.js moduly – nainstaluje potřebné moduly pro Electron
  5. Zástupce – vytvoří ikonu na ploše pro snadné spuštění
💡 Tip: Celý proces instalace trvá 5-10 minut, v závislosti na rychlosti vašeho připojení k internetu.

4️⃣ Aktualizace aplikace

Po instalaci najdete ve složce aplikace soubor update.bat:

  1. Přejděte do složky C:\OPJAK\electron_app
  2. Spusťte update.bat nebo update-windows.bat dvojklikem
  3. Skript synchronizuje větev uloženou v channel-config.json a aktualizuje knihovny
✅ Rychlá aktualizace: Skript stahuje pouze curated obsah z aktivního instalačního kanálu, takže se na počítače kolegů netahají vývojářské soubory.
🧪 Testovací kanál: Pokud vám kolega pošle testovací větev, můžete instalaci jednorázově přepnout příkazem powershell -ExecutionPolicy Bypass -File .\scripts\update_windows.ps1 -Branch windows-install-test. Návrat zpět proběhne stejným způsobem s větví windows-install. Aktualizační logy najdete v logs\update\, start aplikace v testovacím kanálu zapisuje do logs\launcher\.

5️⃣ Řešení problémů

❌ "Python není nainstalován"

  • Zkontrolujte, že Python je nainstalován: otevřete CMD a napište python --version
  • Pokud nevidíte verzi, přeinstalujte Python a zaškrtněte "Add Python to PATH"
  • Po přeinstalaci restartujte počítač

❌ "Git není nainstalován"

  • Stáhněte a nainstalujte Git for Windows
  • Při instalaci ponechte výchozí nastavení
  • Po instalaci restartujte CMD nebo instalátor

❌ "Nepodařilo se nainstalovat Python knihovny"

  • Zkontrolujte připojení k internetu
  • Vypněte dočasně antivirus/firewall
  • Zkuste spustit instalátor jako správce (pravým tlačítkem → "Spustit jako správce")

❌ "Aplikace se nespustí"

  • Zkontrolujte, že máte nainstalovaný Microsoft Excel
  • Přejděte do C:\OPJAK\electron_app\logs
  • Otevřete backend.log a electron.log
  • Pošlete tyto logy kolegovi, který poskytuje podporu

6️⃣ Odinstalace

Pokud chcete aplikaci odinstalovat:

  1. Ukončete aplikaci (pokud běží)
  2. Smažte složku C:\OPJAK
  3. Smažte zástupce z plochy

Python, Node.js a Git můžete ponechat – využijete je i pro jiné aplikace.
